🛡️ 隔离原则
凭证层 (.env) → 只能被服务端读取
适配层 (providers/) → 封装凭证使用
服务层 (services/) → 业务逻辑
接口层 (api/) → 接收业务参数
展示层 (components/) → 仅渲染
