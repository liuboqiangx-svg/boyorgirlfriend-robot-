## 定时早安邮件系统

> 进行中

### 功能说明

每天早上定时给已注册用户发送早安邮件，增加客户黏性。

### 技术方案

- **触发方式**：外部定时服务（cron-job.org）→ 调用 API
- **发送时间**：每天 9:00（北京时间）
- **发送对象**：所有已注册用户
- **邮件特点**：温馨简洁，无需角色元素

### 改动文件

| 文件 | 操作 |
|------|------|
| `lib/db/schema-drizzle.ts` | 新增 email_logs 表 |
| `lib/email/templates/DailyMorningEmail.tsx` | 新建早安邮件模板 |
| `lib/email/sender.ts` | 添加 sendDailyMorningEmail 方法 |
| `app/api/cron/daily-email/route.ts` | 新建 Cron API 端点 |
| `.env.local` | 添加 CRON_SECRET_KEY |

### 执行流程

```
cron-job.org → GET /api/cron/daily-email?key=xxx
    → 查询未发送用户 → 发送邮件 → 记录日志
```

## 消息显示顺序优化

> 已完成

### 问题描述

用户发送表情/消息后，需要等待晓晓回复才显示表情和回复结果

### 解决方案

1. 使用 `flushSync` 确保用户消息立即显示
2. 发送后立即滚动到底部
3. 然后显示"正在输入"动画
