## 如何使用

每次审核完成后：
1. 将本次审核的问题追加到"本次待修复问题"
2. 更新上次审核遗留问题的状态
3. 重新计算修复进度统计
4. 将已修复的问题标记为"已修复"并注明修复日期

状态流转：`待修复` → `修复中` → `已修复` 或 `未修复-原因`
