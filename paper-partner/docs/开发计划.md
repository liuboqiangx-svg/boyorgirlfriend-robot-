### 2. 修复角色 prompt 绑定

**问题**：当前 API 使用 `DEFAULT_CHARACTER.speech_style`，所有角色共用同一 prompt

**解决方案**：
1. 在 `lib/character.ts` 中为每个角色添加 `speech_style` 和 `system_prompt`
2. 修改 `app/api/chat/route.ts`，根据 `characterId` 加载对应角色配置
3. 修改 `generateCharacterReply` 调用，传入正确的角色 prompt

**改动文件**：
- `lib/character.ts` - 添加每个角色的完整 prompt
- `app/api/chat/route.ts` - 根据角色 ID 加载配置

### 3. 配置其他角色语音能力

**问题**：当前只有部分角色有 TTS voiceId 配置

**解决方案**：在 `lib/character.ts` 中为每个角色配置 `voiceConfig`
