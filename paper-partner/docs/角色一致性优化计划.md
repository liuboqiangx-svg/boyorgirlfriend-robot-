## Phase 2: 语音触发器

### 功能描述
只在特定场景触发语音，平时纯文字沟通

### 触发条件

#### 用户触发关键词
| 关键词 | 说明 |
|--------|------|
| 想听 | 用户想听语音 |
| 语音/发语音/声音 | 用户明确要求语音 |
| 说话 | 用户要求说话 |
| 读给我听/念念 | 用户要求朗读 |
| speak/voice/audio | 英文备用 |

#### 情绪自动触发
| 情绪 | 说明 |
|-------|------|
| happy/开心 | 开心时带语音 |
| excited/兴奋 | 兴奋时带语音 |
| clingy/撒娇 | 撒娇时带语音 |
| passionate/热情 | 热情时带语音 |

## Phase 5: 图片生成一致性

### 功能描述
后续所有含角色的图片都以基础头像为参考，保证人物一致性

### 改动位置
- `lib/image/assembler.ts`
- `lib/image/service.ts`

### 改动逻辑
1. 生成图片时，传入 reference_images
2. 使用角色基础头像作为参考
3. image_prompt 描述场景

### 示例
```typescript
generateCharacterImage({
  characterId: "lin-ye",
  scene: "beach",
  reference_images: ["/docs/characters/林野/avatar/front.png"],
  image_prompt: "在海边散步，阳光明媚"
})
```
